# 🐾 Care4Pets


Care4Pets is a web platform designed to help pet owners track their pets' health and allow the community to report cases of animal abuse.

#PT
Care4Pets é uma plataforma web criada para ajudar tutores a acompanhar a saúde de seus pets e permitir que a comunidade registre denúncias de maus-tratos contra animais.

---

# 🚀 Features | Funcionalidades


- Pet registration
- Vaccine tracking
- Feeding records
- Exercise tracking
- Pet health history
- Animal abuse reporting
- NGO dashboard to manage reports

#PT
- Cadastro de pets
- Registro de vacinas
- Registro de alimentação
- Registro de exercícios
- Histórico de saúde do pet
- Registro de denúncias de maus-tratos
- Painel para ONGs analisarem denúncias

---

# 🎯 Project Goals | Objetivos do Projeto

- Help pet owners organize pet health information
- Improve vaccine tracking
- Allow easier reporting of animal abuse
- Support NGOs in managing reports

#PT
- Ajudar tutores a organizar a saúde dos pets
- Melhorar o controle de vacinas
- Facilitar denúncias de maus-tratos
- Ajudar ONGs a acompanhar denúncias

# ▶ How to Run | Como executar

### 1 Clone repository | Clonar repositório

```
git clone https://github.com/moreira-paloma/care4pets.git
```

### 2 Enter project folder | Entrar na pasta

```
cd care4pets
```

### 3 Run the application | Rodar aplicação

```
mvn spring-boot:run
```
